Introducción

En este documento, se reconocerán los conceptos necesarios para lograr una comprensión del contenido, estructuración y funcionalidad en el desarrollo de aplicaciones para el manejo de información sobre plataformas web.Se darán a conocer las nociones básicas del HTML, como por ejemplo las características y generalidades, con una utilización inteligente de las etiquetas HTML más sencillas, para lograr un documento web de calidad suficiente, de acuerdo a los parámetros estipulados.
